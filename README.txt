========================================
  WORD WEBDAV SERVER
  Huong dan su dung
========================================

1. CAI DAT:
   - Chay "install.bat" (1 lan)
   - App se tu dong chay khi Windows khoi dong

2. KIEM TRA:
   - Mo trinh duyet: http://localhost:1900
   - Neu thay trang "WebDAV Server dang chay" = OK!

3. SU DUNG:
   - Mo file tu web (nhan vien click "Chinh sua")
   - Word tu dong mo file
   - Chinh sua xong, nhan Ctrl+S de luu

4. GO BO:
   - Chay "uninstall.bat"

5. CAU HINH:
   - Mo file "config.json" de thay doi:
     + port: Cong (mac dinh 1900)
     + companyApiUrl: URL API cong ty
     + apiEndpoint: Endpoint API
